## 📋 Project Overview

A high-performance backend API for browsing **200,000 products** with fast pagination, category filtering, and data consistency. Built for the CodeVector Labs internship take-home task.

**Live Demo:** [https://product-api-teqn.onrender.com](https://product-api-teqn.onrender.com)

**Interactive API Docs:** [https://product-api-teqn.onrender.com/docs](https://product-api-teqn.onrender.com/docs)

---

## ✨ Features

- ✅ **200,000 Products** - Efficiently seeded using bulk insert
- ✅ **Fast Pagination** - Both offset and cursor-based pagination
- ✅ **Category Filtering** - Filter products by category
- ✅ **Data Consistency** - Cursor pagination prevents duplicates when data changes
- ✅ **Auto Documentation** - Swagger UI generated automatically
- ✅ **High Performance** - Optimized queries with proper indexing
- ✅ **Production Ready** - Deployed on Render.com with PostgreSQL
